# flick [![Build Status](https://travis-ci.org/iilei/flick.svg?branch=master)](https://travis-ci.org/iilei/flick)

> Never ending flick though arrays by using next() / prev()


## Install

```
$ npm install flick
```


## Usage

```js
const flick = require('flick');

flick('unicorns');
//=> 'unicorns & rainbows'
```
